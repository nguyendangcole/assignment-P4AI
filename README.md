# Assignment 1 – Exploratory Data Analysis (EDA)

🚀 **Live Demo Website:** [https://nguyendangcole.github.io/assignment-P4AI/](https://nguyendangcole.github.io/assignment-P4AI/)

## 📖 Overview
This repository contains the source code and documentation for our **Programming for AI & Data Science (CO3135)** group project at HCMUT – VNU-HCM.

Our assignment focuses on a systematic investigation into data patterns and insights through Exploratory Data Analysis (EDA). It encompasses tabular, text, and image data analysis to build a robust foundation for future modeling.

## 🛠️ Run Locally

**Prerequisites:** Node.js installed on your machine.

1. Clone the repository and install dependencies:
   ```bash
   npm install
   ```
2. Run the development server:
   ```bash
   npm run dev
   ```
3. Open `http://localhost:3000` in your browser.

## 🚀 Deployment
This project is automatically deployed to GitHub Pages via GitHub Actions (or manually built via `npm run deploy`).
